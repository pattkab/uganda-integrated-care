About

This open resource hub curates key Ministry of Health documents to assist facilities, districts and implementing partners in operationalizing integrated care within Uganda’s Primary Health Care platform. Host it on GitHub Pages, share with teams, and contribute improvements via pull requests.

Deploy on GitHub Pages

  1. Create a public GitHub repo (e.g., uganda-integrated-care).
  2. Upload all files including the docs/ PDFs.
  3. Go to Settings → Pages, set “Deploy from branch”, pick main and root (or use a docs/ folder).
  4. Your site will be live at https://<username>.github.io/uganda-integrated-care/.
